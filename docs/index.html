User guide

How to use the UI & the library

GitHub Pages setup ↓
  1. Explore quickly in the browser. Use the selector above to toggle between graph presets and scrub the time slider to see interference patterns.
  2. Run your own simulations locally. python main.py --graph grid --nodes 30 --t-max 10 --save-plots plots/ produces fresh plots with your parameters.
  3. Update the presets for this page. Regenerate docs/assets/sample-data.json with the one-liner in the README.
  4. Keep it lightweight. Everything here is static, so it works directly from GitHub Pages without a backend.

Deploy

Host on GitHub Pages

Pages can serve directly from this docs/ folder.

  1. Open Settings → Pages in your repo.
  2. Set Source to Deploy from a branch, then pick your default branch (e.g. main) and /docs as the folder.
  3. Save. Pages will publish to https://<username>.github.io/Quantum-Walks/.
  4. To test locally: python -m http.server 8000 -d docs then visit localhost:8000/.

Presets

What you’re seeing

Grid

3×3 lattice starting from a corner node; symmetry produces even spread.

Random geometric

Spatial edges within radius 0.55 show uneven interference hotspots.

Complete

Fully connected K6 rapidly oscillates between concentration and near-uniformity.

Small-world

Watts–Strogatz (n=10, k=4, β=0.2) mixes locality and shortcuts for varied spread.
